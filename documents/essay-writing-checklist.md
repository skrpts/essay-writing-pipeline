## Essay Writing Checklist

### Planning
- [ ] Understand the assignment brief and marking criteria
- [ ] Research the topic and take structured notes
- [ ] Develop a focused thesis statement
- [ ] Create a detailed outline

### Drafting
- [ ] Write an engaging introduction with clear thesis
- [ ] Develop each body paragraph with topic sentence, evidence, and analysis
- [ ] Address counter-arguments
- [ ] Write a conclusion that synthesises (not just summarises)

### Revision
- [ ] Check argument coherence and logical flow
- [ ] Verify all claims are supported by evidence
- [ ] Ensure smooth transitions between paragraphs
- [ ] Check academic tone is consistent

### Editing
- [ ] Proofread for grammar and spelling
- [ ] Check citation formatting (in-text and reference list)
- [ ] Verify word count is within limits
- [ ] Format according to submission requirements
